About What's Up

What's Up is a demo real-time messaging application developed by MicroIntel for educational and college project demonstration purposes.

🎓 College Project Demo

This project was created mainly for college project demonstration and learning purposes. It showcases how modern messaging applications work using cloud databases and real-time communication technologies.

👨‍💻 Developed By

Developer: MicroIntel
Project: What's Up
Type: Real-Time Messaging App
Status: Demo Version / Under Development

🎯 Why Developed

The project was developed to understand and demonstrate:

  • Firebase Authentication
  • Cloud Database Management
  • Real-time Messaging
  • Modern Mobile UI Design
  • Responsive Web Development
  • Frontend + Backend Integration

✨ Features

  • Google Login
  • Private Messaging
  • Typing Status
  • Reply to Messages
  • Delete Messages
  • Real-time Updates
  • Search Users
  • Firebase Cloud Storage
  • Mobile Responsive UI

⚠ Limitations

  • No group chat support
  • No image/file sharing
  • No voice/video calling
  • No end-to-end encryption
  • Internet required
  • Still under development

⚠ Demo Notice

This application is a demo version made for academic and project presentation purposes. Features and security may continue improving over time.
